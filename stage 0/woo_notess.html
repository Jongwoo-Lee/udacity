HTML - HyperText Markup Language
Includes plain text, HTML, images, videos, music and so on
Browser <-> Internet <-> HTTP <-> Servers

Tags

Structures - <Opening tag> Contents <Closing tag>
<b> bold text </b>
<em> emphasis - italic text </em>
<br> break / skip a line (void - doesn't need a closing tag)
<p> paragraph - line break at the end </p>

Without the closing tag of <em> for example, everything behind <em> is italicized

Making a webpage links
<TAG ATTR = "value"> Contents </TAG> <a> - anchor tag / href = attribute name / value = URL / Contents = click place of the link
Ex) This website is my favorite
--> This website is <a href = "udacity.com">my favorite</a>

Adding images
<img> - images
<img src = "URL" alt = "text">

Inline vs Block
<br>, <b>, <em>, <a>, <strong>, <img> - inline
<p> - block : Makes an individual box around the text - can have width / height

Span vs Div
<span class = "foo"> text </span> - inline / foo is CSS
<div> text </div> - block
